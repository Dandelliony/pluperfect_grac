= Видавництво „Кобза". Вийшла з друку перша серія музичного видавництва у Київі „Кобза". Зміст такий: N1-й—М. Лисенко: „Сумний спів" для фортеп'яна; № 2 —Сениця П: „Буря" для баритона; № 3—Сениця: „Дідусь" для баритона; № 4 —Сениця: „Серед квіток" для тенора; №5—Стеценко К.: „І золотої й дорогої" для бас-баритона; № 6—„Літньої ночі" для скрипки й співу; № 7 — Кошиць А.: „Не співайте, півниченьки", народ, пісня для мішаного хору; № 7—Кошиць А.: „Ой, поїхав Саврадим", нар. пісня для мішан. хору; 9—Хоткевич Гнат: „Одарочка", танець для бандури, й № 10 — Левицький М.: „Як почуєш вночі" — мелодекламація.
