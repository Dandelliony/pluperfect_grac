Як виконується податкове завдання в січні місяці.
---
Завдання по губерні в сумі 4000 карб виконано на 80 відс, окремо по Житомирській окрузі—69 відс, Шепетівській—83 відс, по Коростенеській—23 відс, що дає найслабше виконання в порівнянні з другими округами. По 1-уч м Житоміру—175 відс, 2-уч—232 відс, ІІІ-уч—188 відс та ІУ-уч—23 відс, теж слабко, а пересічно по всьому м Житоміру—145 відс. 
---
Завдання по губерні в сумі 8000 карб виконано на 103 відс. По округах: Житомирській—182 відс, Коростенській—88 від Шепетівській—35 відс. По 1-уч м. Житоміру—50 відс, 2 уч 64 відс, ІІІ уч—45 відсотків, ІУ уч—103 відс, а пересічно по всьому м Житоміру—25 відс. 
Прибутковий податок.
Завдання в сумі 75,000 карб виконано на 72 відс. По Житомірській окрузі—66 відсотків, шепетівській—94 відс, коростеньській—77 відс, по м. Житоміру 57 відсот, з’окрема по 1 уч—50 відс, 2 уч—59 відс, З уч—59 відс, іУ уч 60 відс. Надходження задовольняючі, але за останню тижневку треба збільшити надходження, аби виконати завдання.
Одноразовий податок. 
По губерні січневе завдання за три тижневки виконано на 57 відс: по Житомірській окрузі—87 відс, коростеньській—35 відс, шепетівській—35 відс, по м Житоміру—40 відс, з’окрема по 1 уч—42 відс, 2 уч—25 відс, 3 уч—37 відс, іУ уч—82 відсотків. Надходження незадовольняючі й це загрожує невиконанням завдання за січень м-ць. 
Рентовий податок. 
Завдання по губерні виконано на 6 відсотків, по Житомирській округі—1 відс, коростен—2 відс, шепетівськ—4 відс, по м. Житоміру—7 відс. з’окрема по 1 уч 5 відс, 2 уч—8 відс, 3 уч—7 відс, ІУ уч—10 відс. Надходження по ренті зовсім кволі, катастрофичні. Коли навіть прийняти до уваги, що значна кількість ренти рахується за місцьгоспом та державними установами, все ж надходження й виконання приватними особами менше ніж по других податках. Фінаппаратові необхідно на це звернути пильну увагу, аби через ренту не зірвати взагалі виконання податкового завдання. 
Разом прості податки по губерні виконано на 59 відсотків, з’окрема по Житомірській окрузі—71 відс, коростенській 59 відсотків, шепетівській—65 відс та по м Житоміру—46 відс.
