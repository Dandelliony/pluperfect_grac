 По Житомирський окрузі.
З оперативних даних видно, що на 7 вересня по окремих районах поступило в цілих цифрах. По Черниховському 22 тисячі карб., по Левківському —22 тис. к., по Троянівськом —11 тис. к., по Чуднівськом —13 тис. к., по Міропільському — 7 тис. к., по Андрушівському -12 тис. к., но Городніцькому —16 тис. к., по
Пищівському —22 тис. к., по Звягельськом —27 тис. к., по Барановському—s тис. к., по Пулінському — 19 тис. в., по Янушаільському —6 тис. к., по Любарському —8 тис. карб, і по Солотвінському--7 тис. карбованців. Разом по всій окрузі таким чином внесено гже біля 200 тис. карбованців.
 По Коростенський окрузі.
По Короетенськіп окрузі як свідчать оперативні дні на 1—3 вересня поступило по окремих районах в цілих цифрах. По Олевському —7 тис. к., по Думському- 5 тис. в., по Володарському —20 тис. к., по Барашівському — 20 тис. к., по Фасовській —11,500 к.,
по Словечанському —8 тис. к., по Емільчинському —5; тис. я., но Баварськом —38 тис. к., по Овручському — 22 тис. к., по Народичському —20 тис. карб, та по Коростенеькому — 10 тис. карб. Разом ио окрузі поступило 178 тис. карбованців.
