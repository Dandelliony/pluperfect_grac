По Шепетівський окрузі.

За оперативними даними на 6 вересня по Шепетівській окрузі по окремих районах в цілих цифрах поступило:

По Антонінському —2600 карб., но Березовському —5200 карб., но Гридевськом —14300 карб., по Ст.-Констянтиновському—6500 карб., по Полонському—9500 карб., по Славутському— 11200 карб., по Судилковськом —19000 карб., по Красіловському —2300 карб., по Базалійському — 2900 карб., по Ізаславльському — 43500 к., по Ляховецькому —34000 к.,
по Плужанському —19600 карб., по їеофипольському—llBoo та Аннонпільському—7400 к.
 Разом по окрузі поступило 196 тис карбоваців.
 По всій губернії таким чином поступило вже 572 тисячі карб.  З зазначеннях више цифр видно, що де-які райони дуже пагано викону ють податок. Цим районам треба-вже „раскачаться" пам'ятаючи що 15 вересня—кінець Першого терміну —вже „наносу".
