ЗАКОРДОННІ СОРТИРОВКИ 
заводу бр. ДЕБЕР системи Ідеал і Тріумф ГОДЯТЬСЯ ДЛЯ ОЧИЩЕННЯ РІЖНОГО НАСІННЯ, ЯК ТО: клеверу, люцерни, еспарцету; спеціяльні сортировки «Змійки» для очищення від гороху, віки, відділення віки від овса; полотна для бурячних сортировок «Горок» всіх систем; трієра фабрики Н. ГЕЙДЕНА для очищення пшениці, ячменю від овса; машини для протравлювання зерна від головні й інші машини, потрібні для очищення посівного зерна, ПРОДАЮТЬСЯ Всеукраінським Т-вом НАСІННИЦТВА НА ПІЛЬГОВИХ УМОВАХ ВИПЛАТИ. Кооперативним установам, що мають механичні двигуни, по згоді з товариством, ВЛАШТОВУЮТЬСЯ ГРОМАДСЬКІ ЗЕРНООЧИСТКОВІ ПУНКТИ В ДОВГОЧАСНИЙ КРЕДИТ. Про ціни та умови довідатися в Уповноваженого Всеукраїнського Товариства Насінництва на Полтавщині по адресі м. Полтава, ул. Леніна № 19.
